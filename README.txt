How to Run Luna_Desktop
Step 1: Open Command Prompt
Press Windows key + R ( ⊞ Win + R ).
Type cmd.

Press Enter.
A Command Prompt window will open.

Step 2: Navigate to the Project Folder
You need to move to the folder where the Luna_Desktop file is located.
For example, if it’s inside your Documents folder:
cd Documents

Then go into the Luna_Desktop folder:
cd Luna_Desktop

Or you can do it in one command:
cd Documents/Luna_Desktop

Press Enter after typing the command.

Step 3: Start the Application
Once you're inside the Luna_Desktop folder, type:
npm start


Then press Enter.

Requirements

You must have Node.js and npm installed on your computer.

If you don’t have npm installed or need help setting it up, watch this guide:

https://youtu.be/UYz-9UaUp2E?si=oR8XG0XP3oa8Hdl4
